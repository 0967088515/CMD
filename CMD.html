CMD 指令

CMD 指令及用途

1. ipconfig

顯示當前電腦的 IP 配置信息

常用參數：

  • /all：顯示所有網路適配器的詳細信息。
  • /release：釋放當前的 DHCP 地址。
  • /renew：請求新的 DHCP 地址。
  • /flushdns：清除 DNS 解析緩存。

2. ping

透過向目標地址發送 ICMP 請求，測試與該地址的網路連通性

常用參數：

  • -t：持續發送 ping 請求，直到手動停止。
  • -n：指定發送的請求次數。

3. tracert

追蹤資料包到達目標主機的路徑，顯示每跳路由的信息

常用參數：

  • -d：直接顯示 IP 地址，而不進行域名解析。

4. netstat

顯示網路連接、路由表和網路介面的統計信息

常用參數：

  • -a：顯示所有連接和監聽端口。
  • -n：以數字形式顯示地址和端口號，而不解析域名。
  • -o：顯示與每個連接相關聯的進程ID（PID）。
  • -b：顯示與每個連接相關聯的可執行程序（需要管理員權限）。

5. nslookup

用於查詢域名系統，可以查詢域名的 IP 地址，或查詢與特定 IP 地址相關的域名

6. netsh

配置網路設置，可以進行網路適配器設置、更改防火牆規則、配置網路共享

常用參數：

  • interface：用於配置網路介面的設置。
  • firewall：用於設置防火牆規則。

7. net

用於網路資源管理，包括共享資料夾、打印機、網路用戶

常用參數：

  • user：顯示或管理用戶帳戶。
  • view：查看共享資源。
  • share：管理資料夾共享。

8. route

顯示或修改計算機的路由表

常用參數：

  • print：顯示當前路由表。
  • add：添加路由。
  • delete：刪除路由。

9. arp

顯示或修改計算機的 ARP 緩存，用於查看 IP 地址與MAC 地址之間的關係

常用參數：

  • -a：顯示 ARP 緩存。
  • -d：刪除 ARP 緩存中的某項條目。
  • -s：為某個 IP 地址添加靜態 ARP 緩存條目。

10. telnet

透過 Telnet 協議連接到遠端計連線。
